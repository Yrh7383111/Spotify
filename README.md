# Spotify
Spotify personalized version

## Project Setup
```
pod update
```


## Development Stack
```
Front-end: UIKit, SwiftUI

Back-end: Swift and other CocoaPods packages

Features: Sign In, Album, Browse, Playlist, Search
```
